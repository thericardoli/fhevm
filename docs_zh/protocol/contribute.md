# 贡献指南

有两种方式可以为 FHEVM 做出贡献：

- [提交问题](https://github.com/zama-ai/fhevm/issues/new/choose) 报告错误和拼写错误，或建议新想法
- 通过发送电子邮件至 [hello@zama.ai](mailto:hello@zama.ai) 申请成为正式贡献者。

成为认可的贡献者需要签署我们的贡献者许可协议（CLA）。只有认可的贡献者才能发送拉取请求，因此请务必在操作之前与我们联系！
