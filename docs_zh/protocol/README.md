# 欢迎

**欢迎阅读 Zama 机密区块链协议文档。**\
本文档旨在指导您使用全同态加密（FHE）在任何 L1 或 L2 之上构建机密 dApp。

## 下一步

如果您对 FHE 或 Zama 协议完全陌生，我们建议您首先查看[白皮书](https://docs.zama.ai/protocol/zama-protocol-litepaper)，其中对该协议进行了全面概述。

否则：

🟨 前往[**快速入门**](https://docs.zama.ai/protocol/solidity-guides/getting-started/quick-start-tutorial)学习如何使用 FHEVM 编写您的第一个机密智能合约。

🟨 前往[**Solidity 指南**](https://docs.zama.ai/protocol/solidity-guides)探索加密类型、操作、ACL 和其他核心功能在实践中如何工作。

🟨 前往[**Relayer SDK 指南**](https://docs.zama.ai/protocol/relayer-sdk-guides)构建一个可以加密、解密并与区块链安全交互的前端。

🟨 前往[**区块链上的 FHE**](architecture/overview.md)深入了解架构，并了解加密计算如何在链上和链下组件中流动。

🟨 前往[**示例**](https://docs.zama.ai/protocol/examples)从智能合约示例和 dApp 示例中寻找参考和灵感。

Zama 协议测试网未经审计，不适用于生产环境。**请勿发布任何关键或敏感数据**。对于生产工作负载，请等待主网发布。
