# FHEVM 中文文档

这是 FHEVM（全同态加密虚拟机）文档的中文翻译版本。

## 目录结构

- **protocol/** - 协议架构和核心组件文档
  - architecture/ - 详细的架构组件说明（库、主机合约、协处理器、网关、KMS、中继器）
  - 贡献指南、路线图和加密/解密说明

- **examples/** - 示例和教程
  - 基础示例（FHE 计数器、操作、加密、解密）
  - OpenZeppelin 机密合约
  - 高级示例（密封竞价拍卖）

- **solidity-guides/** - Solidity 开发指南
  - 快速开始教程
  - 配置和设置
  - 类型和操作
  - 访问控制列表（ACL）
  - 解密指南
  - Hardhat 集成
  - 逻辑操作

- **sdk-guides/** - SDK 集成指南
  - 中继器 SDK 概述
  - 初始化和配置
  - 输入注册
  - 用户解密和公共解密
  - Web 应用程序开发
  - 命令行界面

- **operators/** - 操作员文档

## 关于翻译

本翻译旨在帮助中文用户理解和使用 FHEVM 技术。翻译保留了所有代码示例、技术术语和链接的原始格式。

如有翻译问题或建议，请访问原始文档：https://docs.zama.ai/protocol

## 版本

此翻译基于原始英文文档创建。建议定期查看英文文档以获取最新更新。
