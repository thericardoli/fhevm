# 快速入门教程

本教程将指导您快速开始使用 Zama 的**全同态加密 (FHE)** 技术来构建机密智能合约。

## 您将学到什么

在**大约 30 分钟**内，您将从一个基本的 Solidity 合约转变为一个使用 **FHEVM** 的完全机密的合约。以下是您将要做的事情：

1. 设置您的开发环境
2. 编写一个简单的 Solidity 智能合约
3. 将其转换为与 FHEVM 兼容的机密合约
4. 测试您的与 FHEVM 兼容的机密合约

## 先决条件

- 对 **Solidity** 库和 **Ethereum** 的基本了解。
- 对 **Hardhat** 有一定的了解。

{% hint style="info" %}

#### 关于 Hardhat

[**Hardhat**](https://hardhat.org/) 是一个用于编译、部署、测试和调试以太坊智能合约的开发环境。它在以太坊生态系统中被广泛使用。

在本教程中，我们将介绍 FHEVM hardhat 模板，它提供了一种使用 FHEVM 的简单方法。

{% endhint %}
